# Necronormal

A Hytale mod that adds necromancy-themed items and abilities.

## Features

- **Tome of Necromancy** - A magical tome with necromantic powers
- **Soulstone Ore** - A new ore resource
- **Necro Projectiles** - Dark magical projectiles including orbs and roots
- **Summoning Spells** - Charge up and summon creatures

## Installation

1. Place this mod folder in your Hytale mods directory
2. Enable the mod in the game's mod settings

## Contents

### Items
- Tome of Necromancy
- Soulstone Ore

### Projectiles
- Necro Orb
- Necro Roots
